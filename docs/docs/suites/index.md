# 套餐列表

## 概述

NBTI 支持配置多个独立的试题套件，每个套件拥有独立的配置和数据，通过子路由进行区分。

## 套餐索引

套餐索引文件位于 `apps/web/configs/suites/index.json`，包含所有已注册套餐的信息。

## 可用套餐

| ID     | 名称           | 描述                           |
| ------ | -------------- | ------------------------------ |
| `pr01` | 打工人人格测试 | 揭示职场和生活中的隐藏人格特质 |

## 访问套餐

每个套餐通过子路由访问：

- 打工人: `/pr01`
