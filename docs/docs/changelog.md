# 变更日志

## v1.0.0 (2026-04-24)

### 新增

- **多套件支持**：支持配置多个独立的试题套件
- **JSON Schema**：完整的 JSON Schema 定义，支持 AI 友好读取
- **主题系统**：每个套件独立的视觉主题配置
- **国际化**：内置多语言支持
- **VitePress 文档**：完整的开发文档和 Schema 参考

### 功能

- **配置驱动渲染**：同一套页面组件，读取不同 JSON 配置
- **零代码扩展**：新增套件只需添加配置文件
- **平滑迁移路径**：预留 API + 后台管理的升级路径
