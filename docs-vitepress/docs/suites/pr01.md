# pr01 - 打工人人格测试

## 概述

打工人人格测试是一款揭示你在职场和生活中的隐藏人格特质的测评工具。

## 核心概念

本测试围绕以下几个维度展开：

| 维度 | 说明 |
|------|------|
| 工作态度 | 你是积极进取还是随遇而安？ |
| 社交风格 | 在职场中你是社交达人还是独行侠？ |
| 压力应对 | 面对挑战时你的反应方式 |
| 职业追求 | 你最看重职业发展的哪个方面 |

## 测试目的

1. 帮助职场人士了解自己的行为模式
2. 发现潜在的工作风格优势
3. 提供职场人际关系改善建议
4. 指导职业发展规划

## 适用人群

- 初入职场的新人
- 寻求职业突破的职场人士
- 希望了解团队成员风格的领导者
- 对自我认知感兴趣的任何人士

## 计分方式

根据题目设计采用 **维度差值法** 计算，将多个维度组合形成最终的人格类型。
