# DISC 性格测试

## 概述

DISC 是目前最受欢迎的行为风格评估工具之一，通过四个核心维度测量人的行为倾向：支配性（Dominance）、影响性（Influence）、稳健性（Steadiness）和谨慎性（Conscientiousness）。

## 四个维度

| 维度 | 全称 | 特点 | 关键词 |
|------|------|------|--------|
| D | Dominance | 支配型 | 直接、果断、竞争性强 |
| I | Influence | 影响型 | 友好、外向、乐观 |
| S | Steadiness | 稳健型 | 耐心、忠诚、稳定 |
| C | Conscientiousness | 谨慎型 | 精确、分析型、有条理 |

## DISC 图形

DISC 测试结果通常以四象限图表示：

```
         D (支配型)
          ↑
    D     │     I
   15  ←──┼──→  15
          │
  ─────────┼─────────
          │
    S     │     C
   15  ←──┼──→  15
          │
          ↓
         S (稳健型)
```

## 高低分解读

### D（支配型）

**高分特征**：自信、有竞争力、喜欢掌控局面、决策迅速
**低分特征**：不喜欢冲突、倾向于服从他人

### I（影响型）

**高分特征**：善于社交、乐观、喜欢与人互动、善于激励他人
**低分特征**：不喜欢长时间独处、可能过于冲动

### S（稳健型）

**高分特征**：耐心、有同情心、善于倾听、稳定可靠
**低分特征**：不喜欢变化、可能过于顺从

### C（谨慎型）

**高分特征**：精确、注重细节、有条理、善于分析
**低分特征**：可能过于挑剔、决策较慢

## 组合类型

| 组合 | 说明 |
|------|------|
| DI | 果断的影响者 |
| DS | 稳健的支配者 |
| DC | 谨慎的支配者 |
| ID | 有影响力的支配者 |
| IS | 有影响力的稳健者 |
| IC | 分析型的影响者 |
| SD | 稳健的支配者 |
| SI | 稳健的影响者 |
| SC | 分析型的稳健者 |
| CD | 支配的谨慎者 |
| CI | 影响型谨慎者 |
| CS | 稳健型谨慎者 |

## 计分方式

DISC 使用 **百分比法** 计算结果：

```
D% = D得分 / (D + I + S + C) × 100%
I% = I得分 / (D + I + S + C) × 100%
S% = S得分 / (D + I + S + C) × 100%
C% = C得分 / (D + I + S + C) × 100%
```
