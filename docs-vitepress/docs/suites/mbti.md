# MBTI 性格测试

## 概述

MBTI（Myers-Briggs Type Indicator）是最广泛使用的性格评估工具之一，基于卡尔·荣格的心理学理论，将人的性格分为 16 种类型。

## 四个维度

| 维度 | 左端 | 右端 | 说明 |
|------|------|------|------|
| EI | 外向 (E) | 内向 (I) | 精力来源和注意力指向 |
| NS | 直觉 (N) | 实感 (S) | 获取信息的方式 |
| TF | 思考 (T) | 情感 (F) | 做决定的方式 |
| JP | 判断 (J) | 知觉 (P) | 生活态度和方式 |

## 计分方式

MBTI 使用 **维度差值法** 计算结果：

1. 计算每个维度的两端得分
2. 取差值较大的端点作为该维度的结果
3. 组合四个维度的结果形成最终类型代码

例如：
- EI 维度：E 得分 15，I 得分 9 → 结果为 **E**
- NS 维度：N 得分 8，S 得分 12 → 结果为 **S**
- 组合后得到 **ESxx** 类型
